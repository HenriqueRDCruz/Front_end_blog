Como Instalar o MySQL: Um Guia Passo a Passo

Publicado em 23 de Setembro de 2024

O MySQL é um dos sistemas de gerenciamento de banco de dados mais populares do mundo, sendo amplamente utilizado em projetos web e em grandes plataformas. Neste tutorial, vou te guiar pelo processo de instalação do MySQL no Windows e no Linux (Ubuntu).

Instalação do MySQL no Windows

Passo 1: Baixar o MySQL Installer

  • Acesse o site oficial do MySQL e baixe o MySQL Installer for Windows.
  • Escolha entre as versões Web ou Full.

Passo 2: Executar o MySQL Installer

  • Abra o instalador e siga as instruções.
  • Selecione Developer Default para instalar os principais componentes (MySQL Server, Workbench, Shell, etc.).

Passo 3: Configurar o MySQL Server

  • Escolha o tipo de configuração Standalone MySQL Server.
  • Mantenha a porta padrão 3306 e defina uma senha para o usuário root.

Passo 4: Concluir a Instalação

  • Finalize a instalação e inicie o MySQL Workbench ou MySQL Shell.

Passo 5: Testar a Conexão

  • Abra o MySQL Workbench, crie uma nova conexão e insira localhost e a senha de root.
  • Clique em Test Connection para garantir que tudo está funcionando.

Instalação do MySQL no Linux (Ubuntu)

Passo 1: Atualizar o Sistema

  • Atualize o sistema executando os comandos: 
    sudo apt update
sudo apt upgrade

Passo 2: Instalar o MySQL

  • Instale o MySQL com o seguinte comando: 
    sudo apt install mysql-server

Passo 3: Configurar a Segurança do MySQL

  • Execute o comando: 
    sudo mysql_secure_installation
  • Defina a senha do root e aumente a segurança do servidor conforme as instruções.

Passo 4: Iniciar e Verificar o Status do MySQL

  • Verifique o status do MySQL com: 
    sudo systemctl status mysql
  • Se necessário, inicie o serviço com: 
    sudo systemctl start mysql

Passo 5: Acessar o MySQL

  • Use o comando para acessar como root: 
    sudo mysql -u root -p
  • Insira a senha e comece a gerenciar o MySQL pelo console.
  • Conclusão: Agora que você tem o MySQL instalado no Windows ou no Linux, pode começar a gerenciar seus bancos de dados com facilidade. Certifique-se de seguir as práticas recomendadas de segurança para proteger seus dados!
